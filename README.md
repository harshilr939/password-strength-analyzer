# Password Strength Analyzer

A cybersecurity-focused Python tool that evaluates password strength using entropy calculations and character pattern analysis.  
It provides real-time feedback and suggestions to help users create stronger and more secure passwords.

## Features

- Password entropy calculation
- Strength classification:
  - Weak
  - Medium
  - Strong
- Colored terminal output for readability
- Security improvement suggestions
- Regex-based pattern detection

## Technologies Used

- Python
- Regular Expressions (Regex)
- Math (Entropy calculation)
- Colorama (for colored terminal output)
